## 02 - Decisxes de design da v5

01 - A landing passou a adotar visual light para ficar alinhada ao Figma.  
02 - O fundo principal usa `#eef0f7`, com cards brancos e blocos cinza para se aproximar do protótipo.  
03 - A fonte padrão permanece Lexend.  
04 - O header foi simplificado para destacar navegação, CTA e marca HSS.  
05 - A imagem médica do hero foi criada a partir da referência visual enviada para manter proximidade com o layout.  
06 - As seções do Figma foram preservadas: Hero, Benefícios, Como funciona, processo sem/com HSS, Depoimentos, Diferencial, Calculadora e formulário.

## 03 - Decisxes de produto

01 - A calculadora continua usando a lógica de ROI por tempo de credenciamento.  
02 - O painel ADM continua sendo a fonte das premissas de tempo, custo, percentual e visibilidade.  
03 - O usuário público informa valores na calculadora e o sistema usa as tabelas do ADM para calcular ROI.  
04 - Cenários inativos no ADM não aparecem na calculadora pública.  
05 - A versão segue preparada para Cloudflare Pages + D1, mas mantêm fallback local via localStorage para protótipo.
